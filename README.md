# Number-Guessing-Game-Java
A console-based Java application where the system generates a random number and the user attempts to guess it within limited attempts. The program provides feedback, supports multiple rounds, and calculates scores based on performance using core Java concepts.
# Number Guessing Game (Java)

This is a console-based Java application developed as part of a Java Development Internship.

## Features
- Random number generation
- Limited attempts
- Multiple rounds
- Scoring system based on attempts

## Technologies Used
- Java
- OOP concepts
